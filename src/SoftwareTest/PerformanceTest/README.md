## 性能测试课程总体目标

- 使用Jmeter工具进行接口的性能压测
- Jmeter工具掌握
- 针对项目接口进行压测（性能测试的流程和操作过程）
- 分析调优（不是重点）

## 课程大纲

| 章节                                   | 知识点                                                       | 时间    |
| :------------------------------------- | ------------------------------------------------------------ | ------- |
| 第 1 章 性能测试理论知识               | 1.  性能测试概述<br/>2.  性能测试策略<br/>3.  性能测试指标<br/>4.  性能测试流程 | 第1天   |
| 第 2 章 性能测试工具JMeter             | 1.  常用性能测试工具<br/>2.  JMeter环境搭建<br/>3.  JMeter基本使用<br/>4.  JMeter参数化、断言、关联<br/>5.  JMeter录制脚本<br/>6.  JMeter直连数据库<br/>7.  JMeter分布式<br/>8.  JMeter测试报告 | 第2-4天 |
| 第 3 章 项目实战 （微商城）            | 1.  项目介绍和部署<br/>2. 性能测试需求分析<br/>3. 性能测试计划<br/>4. 测试用例设计<br/>5. 测试脚本开发<br/>6. 建立测试环境<br/>7. 执行测试脚本<br/>8. 性能测试监控<br/>9. 性能分析和调优<br/>10. 编写测试报告 | 第5-7天 |
| 第 4 章 Locust框架（代码实现性能测试） | 1. Locust介绍和安装<br/>2. Locust使用                        | 第8天   |

## 课程目标

1.  掌握性能测试理论中常见的性能测试场景、性能测试指标和性能测试流程；

2.  掌握使用JMeter工具发送http接口请求，并掌握断言、关联、参数化等功能；

3.  熟悉性能测试的流程。
