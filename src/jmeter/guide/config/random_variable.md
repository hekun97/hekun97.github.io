# 详解随机变量

## 简单介绍

 Random Variable 用于生成随机**数字字符串**并将其存储在变量中，以备后用

 

#### 相比 __Random 的优势

- __Random 函数也可以生成随机数字字符串，但如果要重复引用的话，需要结合用户自定义变量控件（User Defined Variables）
- 使用 Random Variable 可以生成完随机数字字符串后存储在变量中，可以直接重复引用
