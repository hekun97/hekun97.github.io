什么是 URL 编码

  • URL 编码解码**，又叫百分号编码**，是统一资源定位（URL）的编码方式
  • URL 地址（常说网址）规定了数字，字母可以直接使用，另外一批作为特殊用户字符也可以直接用（ / , : @ 等），剩下的其它所有字符必须通过 %xx 编码处理
  • 编码方法很简单，在该字符ascii码的的16进制字符前面加**%**，如空格字符，ascii码是32，对应16进制是20，那么 urlencode 编码结果是 %20
